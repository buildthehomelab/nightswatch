# Core Concepts

## Tweaks Menu

- Press `` ` `` (backtick) to open the tweaks panel
- Panel lets you change the theme (paper / ink)

## Logs

- Press `l` to open the Dozzle log viewer overlay

## Keyboard Shortcuts

- `j` / `k` — navigate between issues (vim-style)
- `Enter` — expand / collapse focused issue
- `h` or `?` — show keyboard shortcut help overlay
- `r` — manually trigger a refresh/re-check
