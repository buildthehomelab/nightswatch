# Product

## Register

product

## Users

Single homelab operator. Context: glancing at a personal monitor from a desk, occasionally at night or between other tasks. Alert-fatigued; wants signal, not noise. There is no team, no onboarding, no new user.

## Product Purpose

Aggregate anomalies from TrueNAS, Docker, CVEs, and WAN connectivity into a single ranked list. Surface only what needs operator attention. Silence is the default: a healthy state shows nothing but a calm hero phrase. Issues appear ordered by severity; criticals pulse. The absence of content is itself meaningful.

## Brand Personality

Watchful. Spare. Precise. Tone: military-quiet, like a night-watch report. Dry, not playful. Never decorative for decoration's sake.

## Anti-references

- Grafana / generic ops dashboards: no widget soup, no tile grids, no busy status panels
- Modern SaaS (Vercel, Linear light mode, Notion): no cream backgrounds, no rounded card stacks, no marketing softness
- Home Assistant / smart-home tile UIs: no consumer grid, no icon-heavy tiles, no friendly pill buttons everywhere

## Design Principles

1. Silence is signal. The empty state is not a failure state; design the healthy view as carefully as the issue view.
2. Severity speaks first. Color and layout should make critical vs. warning vs. advisory legible in under a second.
3. Information density without clutter. This is an operator tool; data compactness is a feature, not a compromise.
4. Earned color. Color only where it carries meaning: severity, state, or a rare deliberate accent. Never decoration.
5. Night-watch legibility. The interface is read at a glance, often in low-ambient light. High contrast and clear hierarchy matter more than visual novelty.

## Accessibility & Inclusion

WCAG AA minimum (4.5:1 text, 3:1 UI components). Prefers-reduced-motion already respected. Single-user tool, no known additional needs.
